== Description ==
This simple plugin helps the WP website admins retrieve the comments regarding their rental property from Tripadvisor, input them with a shortcode or display them through a widget on their website.

This comment parser plugin is perfect if you:

Have a page regarding your hotel/rooms for rent on the Tripadvisor website
Have lots of comments there
Launch your personal business website for your hotel/rooms for rent
Want the comments from your Tripadvisor page to be shown on your website

== Installation ==

= For automatic installation: =

The simplest way to install is to click on 'Plugins' then 'Add' and type 'WP Comment Parser' in the search field of your website admin panel.

= For manual installation 1: =

1. Login to your website and go to the Plugins section of your admin panel.
2. Click the "Add New" button.
3. Under Install Plugins, click the Upload button.
4. Select the plugin zip file (wp-comment-parser.x.x.x.zip) from your computer then click the "Install Now" button.
5. You should see a message stating that the plugin was installed successfully.
6. Click the Activate Plugin button.

= For manual installation 2: =

1. You should have access to the server where WordPress is installed. If you don't, contact your system administrator.
2. Copy the plugin zip file (wp-comment-parser.x.x.x.zip) to your server and unzip it in some subdirectory.
3. Copy the "WP Comment Parser" folder into the /wp-content/plugins directory of your WordPress installation.
4. Login to your website and go to the Plugins section of your admin panel.
5. Look for "WP Comment Parser" and click Activate.

= Configuration and usage instruction: =

1. After installing and activating the plugin, there will be a new menu item titled “Comment Parser”.
2. Go to the “Comment Parser” section. You will get the plugin menu with just one text field. Enter the URL address of your page on the tripadvisor site, from which you need to retrieve the comments
3. Enter the address in such a format https://www.tripadvisor.<RU>/<YOUR_PLACE>/ and click the ‘Retrieve comments’ button. For example - https://www.tripadvisor.ru/Hotel_Review-g298484-d3539390-Reviews-Khoroshevskaya-Moscow_Central_Russia.html
4. You will receive a short code that you can insert into the Visual Editor.
5. In the Widgets section you will be able to access a new widget - which, like the short code, will import comments from the Tripadvisor to your website.
